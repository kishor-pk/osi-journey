# 🌐 OSI Journey

OSI Journey is an interactive educational game designed to help students and networking enthusiasts learn the OSI (Open Systems Interconnection) model through visualization and hands-on interaction.

## 🎮 About

The game guides players through the seven layers of the OSI model, helping them understand how data travels across a network and what role each layer performs.

It transforms networking concepts into an engaging learning experience.

## 🚀 Features

- Interactive OSI Layer Exploration
- Layer-by-Layer Learning Experience
- Networking Concept Visualization
- Modern User Interface
- Responsive Design
- Progress Tracking

## 📚 OSI Layers Covered

### 7. Application Layer
- User-facing network services
- HTTP, FTP, SMTP

### 6. Presentation Layer
- Data translation
- Encryption and compression

### 5. Session Layer
- Session management
- Connection control

### 4. Transport Layer
- TCP and UDP
- Reliability and flow control

### 3. Network Layer
- Routing
- IP addressing

### 2. Data Link Layer
- Frames
- MAC addressing

### 1. Physical Layer
- Signals
- Cables and transmission media

## 🎯 Learning Objectives

Players will learn:

- The purpose of each OSI layer
- How data moves through a network
- Common networking protocols
- Troubleshooting concepts
- Networking fundamentals used in cybersecurity
